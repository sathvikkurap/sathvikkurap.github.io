Continued Fraction Factorization Method

This project implements the Continued Fraction Factorization method, which uses the continued fraction expansion of square roots to factor large integers.

Key Features:

  • Generation of continued fraction expansions
  • Implementation of convergent finding algorithm
  • Factorization using smooth numbers

Example Usage:


let n = 987654321;
let factors = continued_fraction_factorization(n);
console.log("Factors of " + n + " are: " + factors.join(", "));
