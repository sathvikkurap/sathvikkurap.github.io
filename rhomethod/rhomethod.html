Pollard's Rho Algorithm Implementation

This project implements the Pollard's Rho algorithm for prime factorization, which is particularly efficient for large composite numbers with small prime factors.

Key Features:

  • Efficient factorization of large numbers
  • Implementation of modular exponentiation
  • Utilizes the birthday paradox principle

Example Usage:


let n = 10967535067;
console.log("One of the divisors for " + n + " is " + PollardRho(n));
