Carmichael Number Detector

This project implements an efficient algorithm to detect Carmichael numbers, which are composite numbers that satisfy certain properties similar to prime numbers.

Key Features:

  • Efficient detection of Carmichael numbers
  • Implementation of modular exponentiation
  • Optimization for numbers up to one million

Example Usage:


for(let n = 2; n < 100000; n++) {
    if(is_carmichael(n)) console.log(n);
}
