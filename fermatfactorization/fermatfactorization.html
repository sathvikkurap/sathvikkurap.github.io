Fermat Factorization Method

This project implements the Fermat Factorization method, which is particularly efficient for factoring numbers that are the product of two close factors.

Key Features:

  • Efficient for numbers with close factors
  • Implementation of perfect square detection
  • Optimized search for factor pairs

Example Usage:


let n = 5959;
let [factor1, factor2] = fermat_factorization(n);
console.log("Factors of " + n + " are: " + factor1 + " and " + factor2);
