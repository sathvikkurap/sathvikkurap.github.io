Quadratic Sieve Implementation

This project implements the Quadratic Sieve algorithm, one of the fastest known algorithms for factoring large integers.

Key Features:

  • Efficient factorization of large numbers
  • Implementation of smooth number detection
  • Utilizes factor base for sieving

Example Usage:


let n = 1234567890123456789;
let factors = quadratic_sieve(n);
console.log("Factors of " + n + " are: " + factors.join(", "));
